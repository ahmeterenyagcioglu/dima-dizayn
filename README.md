Dima Dizayn & Organizasyon
İzmir Bergama'da faaliyet gösteren Dima Dizayn & Organizasyon firması için geliştirilmiş, modern, hızlı ve SEO odaklı kurumsal web sitesi.

Teknolojiler:
Framework: Next.js 14 (App Router)
Styling: Tailwind CSS
Language: TypeScript
Deployment: Vercel
Domain: GoDaddy üzerinden dimadizayn.com

Özellikler
Modern UI/UX: Mobil öncelikli, şık ve kullanıcı dostu arayüz.
Gelişmiş Galeri: Etkinlik türlerine göre filtrelenebilir, yüksek performanslı görsel galerisi.
SEO & Performans: * WebP formatında optimize edilmiş görseller.
Dinamik sitemap.xml ve robots.txt yönetimi.
Google Search Console doğrulaması tamamlanmış altyapı.
İletişim: WhatsApp entegrasyonu ve dinamik iletişim formları.

Projeyi kendi yerel makinenizde çalıştırmak için:
Bash
# Bağımlılıkları yükleyin
npm install
# Geliştirme sunucusunu başlatın
npm run dev

Notlar:
Proje Vercel üzerinde otomatik dağıtıma (CI/CD) bağlıdır. main branch'ine yapılan her push doğrudan canlıya yansır.
Statik önbellek sorunlarını önlemek için kritik sayfalarda force-dynamic ayarı kullanılmıştır.
